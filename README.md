## Deskripsi Proyek
Aplikasi **Katalog Novelist** adalah platform manajemen katalog novel berbasis web. Aplikasi ini memungkinkan pengguna untuk mendata koleksi novel mereka secara digital.

## Fitur Utama
- Menambah koleksi novel baru beserta unggah foto sampul.
- Menampilkan daftar koleksi novel dalam tabel yang rapi.
- Mengubah informasi novel yang sudah ada.
- Menghapus data novel dari rak buku digital.
